# Marketing-Insights-For-E-Commerce-Company
📁 Project Overview

This project provides end-to-end marketing analytics for an e-commerce company using transactional, customer, marketing, and tax data. The goal is to extract insights, build predictive models, perform customer segmentation, and create dashboards for monitoring KPIs.

Using multiple datasets from 2019, the project uncovers customer behavior patterns, revenue drivers, and product performance, helping shape strategic marketing decisions.

🎯 Business Objectives
1. Revenue Calculation

Compute invoice value for each transaction:

Invoice Value = ((Quantity * Avg_Price) * (1 - Discount_pct) * (1 + GST)) + Delivery_Charges
2. Exploratory Data Analysis (EDA)

Monthly customer acquisition

Monthly retention & churn

Revenue from new vs existing customers

Impact of discounts on sales

KPI analysis (Revenue, AOV, Transactions, Customers, Quantity)

Trends: Category-wise, weekly, monthly, seasonal

Marketing spend vs revenue correlation

Product-wise frequency & popularity

Monthly tax, delivery charges, and marketing spend %

3. Customer Segmentation
Heuristic Segmentation

RFM (Recency, Frequency, Monetary)

Value-based segmentation: Premium, Gold, Silver, Standard

Machine Learning (K-Means)

Cluster profiling

Marketing strategy for each segment

4. Customer Lifetime Value (CLV) Prediction

Define target classes: Low / Medium / High value customers

Build classification models to predict CLV group

Evaluate model performance

5. Market Basket Analysis

Identify frequently bought-together products

Generate association rules

Cross-sell recommendations

Predict customer return category:

60–90 days

90+ days

7. Cohort Analysis
